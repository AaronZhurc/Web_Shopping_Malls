### web商城 - Web Shopping Malls 
*本项目是web课程设计课程提交的课程设计*  
*This project is a course design submitted for the web-development course*
#### 声明 - Announcement
本程序中使用的如Quin头像、Nintendo Switch、PlayStation、Xbox图片仅做展示使用  
本程序前端设计参考了2020年下半年京东商城的前端设计  
本程序实际上是web软件课程设计，并不太涉及到计算机网络相关

This program uses such as Quin avatar, Nintendo Switch, PlayStation, Xbox images for display purposes only.  
The front-end design of this program refers to the front-end design of the Jingdong Mall in the second half of 2020.  
This program is actually a web software course design, and does not involve too much computer network-related.  
#### 开发环境 - Development Environment
IntellJ IDEA,MySQL,Apache，版本记不清了.(Sorry, can't remember the version.)
